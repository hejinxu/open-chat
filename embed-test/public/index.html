Welcome to ExampleSite

This is a demo page showing the Open Chat embeddable widget in action.

The floating chat button should appear in the bottom-right corner. Click it to open the chat window. Drag the icon to reposition it.

Embed Configuration

window.openChatConfig = {
  baseUrl: 'http://localhost:3000',
  apiKey: 'REPLACE_WITH_YOUR_API_KEY',
  iconUrl: 'https://api.dicebear.com/9.x/bottts/svg?seed=openchat',
  windowTitle: 'AI Assistant',
  theme: 'tech-blue',
  windowSize: {
    width: 420,
    height: 640,
    minWidth: 320,
    minHeight: 400,
    maxWidth: 800,
    maxHeight: 900,
  },
  headerStyle: {
    backgroundColor: '#1C64F2',
    textColor: '#ffffff',
    iconColor: '#ffffff',
    buttonHoverBg: 'rgba(255,255,255,0.15)',
  },
}
Setup: Replace REPLACE_WITH_YOUR_API_KEY with an API key generated in the admin panel.
Built-in icons (14): robot · bot · chat · sparkle · headset · message · brain · wand · rocket · puzzle · eye · code · gear. Set icon: 'chat' to switch, or use iconUrl for custom images.

Feature Tests

Use this page to verify the following embed behaviors:

  • Floating button: Should appear at bottom-right with a robot icon
  • Drag: Click and drag the button to reposition it on screen
  • Open: Click the button (without dragging) to open the chat window
  • Window drag: Drag the title bar to move the chat window
  • Resize: Drag the bottom-right corner handle to resize the window
  • Close: Click the × button to close and restore the floating icon
  • Mobile: Use DevTools device emulation to test fullscreen mode
