# 第三方应用集成指南

> 面向希望在自己的 Web 应用中嵌入 open-chat 对话功能的开发者。
> 前提：open-chat 服务已部署并正常运行。

## 快速开始

### 1. 获取 Embed Token

在 open-chat 服务器上运行 seed 脚本：

```bash
node webapp/scripts/seed-embed-tokens.mjs
```

输出示例：

```
Inserted token: "默认嵌入 Token" (sk-embed-default-a1b2c3d4)
Done.
```

记下输出的 token 值。也可以从 JSON 文件批量导入：

```bash
node webapp/scripts/seed-embed-tokens.mjs --file=tokens-data.json
```

JSON 文件格式：

```json
[
  {
    "id": "tok-001",
    "name": "客户A官网",
    "description": "客户A官网首页嵌入",
    "token": "sk-abc123def456",
    "allowed_agent_ids": ["agent-general"],
    "is_enabled": true
  }
]
```

> `allowed_agent_ids` 设为 `["*"]` 表示允许使用全部智能体。

### 2. 在网页中引入

```html
<!-- 1. 配置 (放在 <script> 引入之前) -->
<script>
window.openChatConfig = {
  baseUrl: 'http://your-server.com',
  token: 'sk-embed-default-a1b2c3d4',
}
</script>

<!-- 2. 引入脚本 -->
<script src="http://your-server.com/embed.min.js" defer></script>
```

> **注意**：如果 open-chat 部署在子路径下（`NEXT_PUBLIC_BASE_PATH=/chat`），`baseUrl` 和脚本 `src` 都需要加上前缀：
> ```html
> baseUrl: 'http://your-server.com/chat'
> <script src="http://your-server.com/chat/embed.min.js" defer></script>
> ```

### 3. 完成

刷新页面，右下角会出现浮动对话按钮。点击即可开始对话。

## 配置参数说明

### 必填参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `baseUrl` | `string` | open-chat 服务器地址，不含尾部 `/`。示例：`'http://localhost:3000'` |
| `token` | `string` | Embed Token，由 open-chat 管理员分配。校验不通过则所有 API 返回 401 |

### 可选参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `agentId` | `string` | 无 | 预设智能体 ID。不填则使用默认智能体 |
| `icon` | `string` | `'robot'` | 内置图标名称（共 14 个） |

### 内置图标

| 名称 | 图标 | 场景 |
|------|------|------|
| `robot` | 机器人 | 通用 AI 助手（默认） |
| `bot` | 双屏机器人 | 备选风格 |
| `chat` | 聊天气泡 | 客服 / 对话 |
| `sparkle` | 星光 | 智能助手 |
| `headset` | 头戴耳机 | 人工客服 |
| `message` | 信封 | 消息中心 |
| `brain` | 大脑 | 智能分析 / 推理 |
| `wand` | 魔法棒 | AI 创作 / 生成 |
| `rocket` | 火箭 | AI 效率提升 |
| `puzzle` | 拼图 | 解题 / 逻辑推理 |
| `eye` | 眼睛 | 视觉分析 / 识别 |
| `code` | 代码 | 编程助手 |
| `gear` | 齿轮 | 系统管理 |

> 来源：`webapp/public/images/embed-icons/` 目录。设置 `iconUrl` 可覆盖内置图标。
| `iconUrl` | `string` | 无 | 自定义图标 URL。设置后覆盖 `icon` |
| `windowTitle` | `string` | `'AI 助手'` | 窗口标题栏文字 |
| `theme` | `string` | `'light'` | 主题：`light` / `dark` / `tech-blue` |
| `locale` | `string` | `'zh-Hans'` | 语言 |
| `inputs` | `object` | `{}` | 预设参数，填入智能体的 prompt_variables |

### 窗口尺寸

```javascript
windowSize: {
  width: 420,        // 初始宽度 (px)，默认 400
  height: 640,       // 初始高度 (px)，默认 600
  minWidth: 320,     // 最小宽度，用户拖拽 resize 下限
  minHeight: 400,    // 最小高度
  maxWidth: 800,     // 最大宽度
  maxHeight: 900,    // 最大高度
}
```

窗口尺寸会持久化到 localStorage (`open_chat_embed_window_size`)，下次打开恢复。

### 标题栏样式

```javascript
headerStyle: {
  backgroundColor: '#1C64F2',           // 标题栏底色
  borderColor: 'rgba(255,255,255,0.2)',  // 标题栏底边
  textColor: '#ffffff',                 // 标题文字色
  iconColor: '#ffffff',                 // ☰ 和 × 图标色
  buttonHoverBg: 'rgba(255,255,255,0.15)',  // 按钮悬停底色
}
```

不填则按 `theme` 取对应默认值。

### 浮动按钮样式

```javascript
bubbleStyle: {
  backgroundColor: '#1C64F2',  // 按钮底色。设置了 iconUrl 或使用内置 icon 时仅作 fallback
}
```

### 完整示例

```html
<script>
window.openChatConfig = {
  baseUrl: 'https://chat.example.com',
  token: 'sk-embed-abc123',
  agentId: 'agent-general',
  icon: 'chat',
  windowTitle: '智能客服',
  theme: 'tech-blue',
  locale: 'zh-Hans',
  windowSize: {
    width: 420,
    height: 600,
    minWidth: 320,
    minHeight: 400,
    maxWidth: 800,
    maxHeight: 900,
  },
  headerStyle: {
    backgroundColor: '#4F46E5',
    textColor: '#ffffff',
    iconColor: '#ffffff',
    buttonHoverBg: 'rgba(255,255,255,0.15)',
  },
  inputs: {
    user_name: '访客',
  },
}
</script>
<script src="https://chat.example.com/embed.min.js" defer></script>
```

## 功能说明

| 功能 | 说明 |
|------|------|
| **浮动按钮** | 右下角 52px 圆形按钮，可拖动到页面任意位置，位置自动保存 |
| **拖拽窗口** | 拖动标题栏可移动弹窗位置 |
| **调整大小** | 右下角 resize handle 可拉宽拉高，受 min/max 约束 |
| **关闭窗口** | 标题栏 × 按钮关闭弹窗，恢复浮动图标 |
| **侧边栏** | 标题栏 ☰ 按钮打开对话历史侧边栏 |
| **智能体切换** | 窗口内支持切换智能体（需 token 的 `allowed_agent_ids` 包含对应 agent） |
| **语音输入** | 支持浏览器语音识别和 Whisper 语音识别 |
| **文件上传** | 支持上传文件/图片作为对话上下文 |
| **主题切换** | 窗口内支持切换 light / dark / tech-blue 主题 |

## 移动端行为

屏幕宽度 ≤ 768px 时自动切换为全屏模式：
- 弹窗覆盖整个屏幕
- 隐藏 resize handle
- 标题栏不可拖拽
- 关闭按钮仍然可用

用户可以在移动端正常对话，关闭后恢复浮动按钮。从移动端切换到桌面端时，窗口自动恢复为预设尺寸。

## 位置持久化

| 存储 Key | 内容 |
|----------|------|
| `open_chat_embed_button_pos` | 浮动按钮位置 `{ x, y }` |
| `open_chat_embed_window_size` | 窗口尺寸 `{ w, h }` |

均存储在用户浏览器的 localStorage 中，刷新页面后保持不变。

## 常见问题

### 窗口没有显示 / 401 错误

检查：
1. `token` 是否正确（和 seed 脚本输出一致）
2. `baseUrl` 是否能通（浏览器访问 `baseUrl/embed` 看是否返回页面）
3. 控制台是否有 `[open-chat-embed]` 错误日志

### 浮动按钮位置不对

- 初始位置：右下角，距边缘 24px
- 拖动后会保存新位置
- 如果位置异常，在 DevTools 中删除 `open_chat_embed_button_pos` 后刷新页面复位

### 弹窗中对话内容不全

iframe 加载需要时间，确保 open-chat 服务器正常运行。打开浏览器 DevTools Network 面板查看 `/api/chat-messages` 请求是否返回 200。

### 跨域问题

如果集成网站和 open-chat 服务器不同域名：
- `embed.min.js` 和 `/embed` 页面必须同域（iframe 同源策略）
- 如果必须跨域，需要 open-chat 服务端配置 CORS 和 `SameSite=None; Secure` cookie
