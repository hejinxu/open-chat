## 主题系统架构概述

当前主题系统由以下核心组件组成：

```
config/theme.ts          # 主题配置定义
hooks/use-theme.ts       # 主题状态管理 Hook
app/components/theme-provider/  # 主题 Context 提供器
app/components/theme-toggle-button/  # 主题切换按钮
app/styles/globals.css # 全局样式
tailwind.config.js    # Tailwind 配置
```

## 步骤 1：更新主题配置

首先，在 `config/theme.ts` 中添加新主题：

```typescript
// config/theme.ts

export const THEME_MODES = {
  LIGHT: 'light',
  DARK: 'dark',
  SYSTEM: 'system',
  // 添加您的新主题: 'your-new-theme', // 新增
} as const

export type ThemeMode = typeof THEME_MODES[keyof typeof THEME_MODES]

export const isValidThemeMode = (mode: string): mode is ThemeMode => {
  return Object.values(THEME_MODES).includes(mode as ThemeMode)
}

export const DEFAULT_THEME_MODE: ThemeMode =
  (process.env.NEXT_PUBLIC_DEFAULT_THEME as ThemeMode) || THEME_MODES.LIGHT
```

### 环境变量配置

在 `.env.local` 或 `.env.local.example` 中添加新主题选项：

```env
NEXT_PUBLIC_DEFAULT_THEME=your-new-theme
```

## 步骤 2：扩展主题管理 Hook

在 `hooks/use-theme.ts` 中添加新主题支持：

```typescript
// hooks/use-theme.ts

import { ThemeMode, THEME_MODES, DEFAULT_THEME_MODE, isValidThemeMode } from '@/config/theme'

// 如果需要特殊的主题初始化逻辑，在 useTheme Hook 中添加：

const useTheme = () => {
  const [theme, setThemeState] = useState<ThemeMode>(() => {
    if (typeof window === 'undefined') return DEFAULT_THEME_MODE
    const saved = localStorage.getItem('theme-mode')
    if (saved && isValidThemeMode(saved)) return saved
    return DEFAULT_THEME_MODE
  })

  // 添加您的新主题相关逻辑
  // ...
}
```

## 步骤 4：添加 Tailwind 主题支持

如果需要在 Tailwind 中使用新主题的颜色，可以使用以下两种方式：

### 方式一：通过配置文件（可选）

```javascript
// tailwind.config.js

module.exports = {
  darkMode: 'class',
  theme: {
    extend: {
      colors: {
        // 您的主题专属颜色
      }
    }
  }
}
```

### 方式二：使用 Tailwind 类名（推荐）

直接在组件中使用标准的 Tailwind 类名前缀方式：

```tsx
<div className="bg-white your-new-theme:bg-blue-900">
  内容
</div>
```

## 测试和验证

### 1. 测试清单

- [ ] 在 `.env.local` 中设置默认主题为新主题，验证页面初始化正确加载
- [ ] 测试主题切换功能
- [ ] 测试 localStorage 保存功能
- [ ] 深色/浅色模式切换
- [ ] 跟随系统功能
- [ ] 您的新主题完整显示

### 2. 常见问题排查

**问题：页面没有正确应用主题颜色？

**解决方案：**
1. 检查 `tailwind.config.js` 中的 `darkMode: 'class'
2. 检查是否正确设置了 `html` 元素上的类名
3. 检查组件是否使用了正确的类名前缀

**问题：主题切换不生效？**

**解决方案：**
1. 检查 `useTheme` Hook 中的主题切换逻辑
2. 检查 ThemeProvider 是否正确包裹了应用

---

## 最佳实践

1. **保持一致性**：始终使用标准的 Tailwind 类名前缀方式
2. **可访问性**：确保新主题满足 WCAG 2.1 对比度要求
3. **性能**：避免过度使用内联样式，尽量使用 Tailwind 类名
4. **测试**：在不同设备和浏览器上测试新主题
